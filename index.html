Essence of Agents

So... what IS an "agent", actually?

The word is everywhere; a real definition is nowhere. So let's pin it down by building one - and it turns out an agent is a while loop around a model. Build the loop, harden it for production, open the model inside it - then see what the frontier is doing to all three. Cortex and Bit walk you through.
