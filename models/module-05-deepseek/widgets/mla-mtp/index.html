DeepSeek's moves: MLA and MTP

Module 3 left a wall: the KV cache is the memory bill for long context, and it is huge. DeepSeek-V3 runs 128k context cheaply anyway - with two targeted moves. MLA (Multi-head Latent Attention) shrinks the cache (Module 3's wall); MTP (Multi-Token Prediction) speeds up decoding (Module 0's one-pass-per-token loop). Each attacks ONE resource - the skill is spotting which.
