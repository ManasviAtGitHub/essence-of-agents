The compiler you can talk to

A compiler turns a spec into a runnable thing - deterministically, and it fails LOUDLY. An LLM is stochastic and speaks fuzzy English. This capstone bolts one to the other: the model is the front-end that lowers a sentence into a checkable plan, and everything downstream is deterministic and verified. The trick that makes a stochastic model behave like a compiler lives inside Module 0 - the sampler.
