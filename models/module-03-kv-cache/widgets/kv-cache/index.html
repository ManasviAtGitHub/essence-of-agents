The KV cache: why context costs memory

The first token takes a beat; the rest stream fast. Long context costs money even when the text is the same. Both facts come from one thing: to look back (Module 2), the model must REMEMBER every past token's keys and values. That store is the KV cache - and it is why "context window" really means memory budget.
